# 📊 Data Analytics Project: Shipping & Revenue Optimization (Northwind DB)

This repository contains the end-to-end data pipeline and SQL queries developed to extract, clean, and consolidate logistics and commercial data from the *Northwind* relational database. 

The primary objective of this project was to transform complex transactional data into actionable business insights, serving as the cloud data source for an interactive executive dashboard built in *Tableau Public*.

## 🛠️ Tech Stack & Tools
* *Database Management:* Microsoft SQL Server
* *Query Language:* T-SQL (Advanced querying, multiple INNER JOINs, and Data Aggregations)
* *Data Pipeline & Cloud Storage:* Google Sheets (CSV integration)
* *Data Visualization / BI:* Tableau Public

## 📁 Project Structure
* Northwind_Queries.sql: Source code containing the SQL queries implemented to isolate shipping carrier performance, revenue per sales agent, and profitability by product category.
* *Interactive Dashboard:* [👉 Click here to view the live Dashboard on Tableau Public] (https://public.tableau.com/views/Northwind_17842562162540/NorthwindDashboard?:language=es-ES&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link))

## 📈 Key Business Insights
* *Logistics & Supply Chain Efficiency:* Identified that a single shipping partner handles 40.07% of the total distribution volume. This insight allows management to strategically evaluate carrier dependency and renegotiate contract terms.
* *Commercial Performance:* Centralized data for sales agents (Revenue per Salesperson) and margins by category, eliminating information silos and accelerating executive decision-making.
